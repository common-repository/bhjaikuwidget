Add up to 9 sidebar widgets to display your Jaiku Updates

== Description ==

= English =

The plugin add up to 9 sidebar widgets to display the Jaiku updates of your different accounts.

= French =

Ce plugin ajoute jusqu'à 9 widgets à votre barre latérale pour afficher les dernières mises à jour de vos différents comptes Jaiku.

== Installation ==

= English =

1. Unzip into your `/wp-content/plugins/` directory. If you're uploading it make sure to upload
the top-level folder. Don't just upload all the php files and put them in `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Presentation » Widgets' submenu and add the 'Jaiku' widgets to your sidebar
4. Configure your widgets (don't forget to indicate how many widgets you want and your Jaiku's account name)
5. Nothing else !

= French =

1. décompressez l'archive dans votre répertoire '/wp-content/plugins'. Si vous uploadez, soyez sûr d'uploader le dossier racine. N'envoyez que les fichiers php dans '/wp-content/plugins'.
2. Activez le plugin depuis le menu 'Plugins' de votre console WordPress
3. Allez dans le sous-menu 'Thème » Widgets' et ajoutez les widgets 'Jaiku' à votre sidebar
4. Configurez les widgets (n'oubliez pas de préciser combien de widgets vous désirez et votre identifiant Jaiku)
5. Rien de plus
